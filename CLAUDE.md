# FlashPWA - Irish Language Learning App

## Project Overview
This is an Ionic Vue 3 Progressive Web App (PWA) for Irish language learning, focusing on flashcard-based learning with verb conjugations and adjectives.

## Tech Stack
- **Framework**: Vue 3 with TypeScript
- **UI Framework**: Ionic Framework v7
- **Build Tool**: Vite 5
- **PWA Support**: vite-plugin-pwa
- **Mobile**: Capacitor 5
- **Testing**: Vitest (unit), Cypress (e2e)
- **Linting**: ESLint with Vue plugin

## Project Structure
```
/src
  /components - Vue components (AdjectiveContainer, ExploreContainer)
  /views - Page components (Tab1Page, Tab2Page, Tab3Page, TabsPage)
  /router - Vue Router configuration
  /theme - CSS variables and theming
  App.vue - Root component
  main.ts - Application entry point
/public
  /samples - JSON data files for forms (1000+ files)
/python - Python scripts for data generation
/tests - Unit and e2e tests
```

## Available Commands
```bash
npm run dev          # Start development server
npm run build        # Type-check with vue-tsc and build for production
npm run preview      # Preview production build
npm run test:unit    # Run unit tests with Vitest
npm run test:e2e     # Run e2e tests with Cypress
npm run lint         # Lint code with ESLint
```

## Key Configuration
- **Base URL**: `/flashpwa/` (configured in vite.config.ts)
- **TypeScript**: Strict mode enabled
- **Path Alias**: `@/` maps to `./src/`
- **PWA**: Auto-update registration with custom manifest

## Data Files
The app uses JSON files in `/public/samples/` containing Irish language forms data. These appear to be generated by Python scripts in the `/python/` directory.
